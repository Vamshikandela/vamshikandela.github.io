01 — About

From Python fundamentals to autonomous agents

Every project follows the same discipline: understand the data, design the retrieval, let the agent reason, then measure what actually improved.

I'm Kandela Vamshi, a Jr. Python Developer based in Hyderabad, currently deep in an Agentic AI internship at Innomatics Research Labs — building Generative AI applications with Python, LangChain, LangGraph, and FastAPI.

My work centers on Retrieval-Augmented Generation (RAG) pipelines and multi-step reasoning agents that pull from real document knowledge sources instead of guessing. I care about the parts most demos skip: context precision, faithfulness, and what happens when the model doesn't know the answer.

Alongside GenAI, I build with Django — REST APIs, MVT architecture, and full-stack integration — so the agents I design don't stay in a notebook, they ship as working applications.
